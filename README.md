# Your own game box photos

Drop a photo here named to match each game, and the game-selection tiles will use it automatically instead of the fallback icon:

- rook.jpg
- handfoot.jpg
- phase10.jpg
- gnoming.jpg
- sequence.jpg
- backwards8.jpg
- custom.jpg   (used for the "Custom Game" tile)

Notes:
- .jpg is what the app looks for by default. If your photos are .png, tell Claude and it's a one-line change to look for that extension instead.
- Any image works, but something close to a square crop (roughly 1:1) will look best in the tile.
- This README itself doesn't do anything — delete it once you've dropped in real photos, or just leave it, it won't affect the app.
